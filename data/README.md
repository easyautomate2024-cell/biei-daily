# 観測台帳

「今日の美瑛」のスコア(予測)と、実際の見え方(目撃情報)を突き合わせるための台帳です。
スコアの係数はこの記録を根拠に調整します — [スコアの仕組み](../logic.html)に書いた「外れたら直す」の実体がこのファイルです。

## observations.csv の列

| 列 | 値 | 説明 |
|---|---|---|
| 日付 | YYYY-MM-DD | 観測日 |
| 池の色 | 青 / 微妙 / 白濁 / (空欄) | 実際の青い池の色。不明なら空欄 |
| 霧 | 出た / 薄い / なし / (空欄) | その朝、丘に霧・雲海が出たか。不明なら空欄 |
| 池スコア | 0-100 | その日にサイトが出していた青い池スコア |
| 霧スコア | 0-100 | その朝のサイトの霧・雲海チャンス |
| 情報源 | 現地 / IG / DM / SNS / カメラ | 目撃情報の出どころ |
| メモ | 自由記述 | 例: 雨後3日目、雲海は下富良野側のみ 等 |

## 運用方針

- **全日埋める必要はありません。** まばらでも、件数が貯まれば係数調整の根拠になります
- **スコアが極端な日(80以上 / 25以下)を優先して確認**します。的中検証として最も情報量が多いのがこの両端です
- 目撃情報の提供は Instagram [@biei_explorer](https://www.instagram.com/biei_explorer/) のDMで歓迎しています
