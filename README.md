# 今日の美瑛 — データで読む、美瑛のいま

美瑛を中心とした個人サイトの第1弾。コンテンツ:

- **今日の青い池スコア** / **明朝の霧・雲海チャンス**(Open-Meteoから毎日自動算出)
- **光の時刻表**(日の出・日の入り・撮影ゴールデンタイム)
- **丘の色ごよみ**(日付連動・年10期の畑の色カレンダー)
- **十勝岳 火山情報**(気象庁の噴火警報JSONをリアルタイム表示)
- **美瑛のきろく**(管理人撮影のギャラリー+Instagram @biei_explorer 導線)

サーバー不要の静的サイト(HTML + CSS + JS のみ)。ブラウザが直接気象APIを
呼ぶため、ホスティング費用も保守もゼロで運用できる。

## スコアのロジック(app.js)

- **青い池スコア**: 直近48時間の降水(濁り) / 日中9〜15時の雲量(光) /
  風速(水面の波立ち) / 当日の降水予報 から 0〜100 で算出
- **霧・雲海チャンス**: 夜間の雲量(放射冷却) / 明け方の風 / 明け方の湿度 /
  気温と露点の差 から算出。放射冷却型の朝霧を想定
- 係数は経験則ベースの初期値。実際の的中を見ながら調整していく
