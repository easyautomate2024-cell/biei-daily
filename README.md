# 今日の美瑛 — データで読む、美瑛のいま

**▶ サイトはこちら: https://easyautomate2024-cell.github.io/biei-daily/**

北海道・美瑛町の「いま」を、感想ではなくデータで案内する非公式の個人サイトです。
青い池は今日青いのか。明日の朝、丘に霧は出るのか。麦はいつ金色になるのか——
気象データから毎日自動で算出し、計算式ごと公開しています。

## コンテンツ

| 機能 | 内容 |
|---|---|
| 今日の青い池スコア | 直近の雨(濁り)・雲量(光)・風(水面)から見頃を0〜100で推定 |
| 明朝の霧・雲海チャンス | 放射冷却型朝霧の条件(晴れた夜・無風・高湿度)の充足度 |
| 光の時刻表 | 日の出・日の入り・撮影ゴールデンタイム |
| 丘の色ごよみ | 畑の色の移ろいを年10期で案内する日付連動カレンダー |
| 麦秋メーター | 積算温度の平年比から「丘が金色になる日」を予測 |
| 十勝岳 火山情報 | 気象庁の噴火警報をリアルタイム表示 |
| 直売所の歩き方 | 無人販売所の作法ガイド+公式直売スポットの地図(個人宅前の販売所の場所は掲載しない方針) |

スコアは「予報」ではなく**条件充足度の推定**です。すべての計算式は
[スコアの仕組み](https://easyautomate2024-cell.github.io/biei-daily/logic.html)で公開しており、
実際の見え方と照らして係数を継続的に調整しています。

## データ出典

- 気象予報・過去データ: [Open-Meteo](https://open-meteo.com/)(CC BY 4.0)
- 火山情報: [気象庁](https://www.jma.go.jp/)
- 写真: すべて管理人が美瑛で撮影([Instagram @biei_explorer](https://www.instagram.com/biei_explorer/))

## 技術構成

HTML / CSS / JavaScript のみの静的サイト(ビルドなし・サーバーなし)。
ブラウザが直接気象APIを呼ぶため、ホスティングは GitHub Pages だけで完結しています。

ローカルで動かす場合:

```
python -m http.server 8080 --directory .
```

開発メモ: `app.js` を変更したら `index.html` の `<script src="app.js?v=N">` の N を上げてください(キャッシュ対策)。
