Methodology

スコアの仕組み

このサイトのスコアは、すべて計算式を公開しています

Data Source

使っているデータ

Open-Meteo(無料の気象予報API)から、 白金青い池付近(北緯43.489度・東経142.621度、標高約500m)の予報値を取得しています。 使う項目は次の6つだけです。

  • 雲量(1時間ごと・%)
  • 降水量(1時間ごと・mm)— 過去2日分も取得
  • 風速(1時間ごと)
  • 湿度・気温・露点(1時間ごと)

カメラやセンサーは使っていません。つまりこのスコアは「実際に見た色」ではなく「色が出やすい気象条件がどれだけ揃っているか」です。

Blue Pond Score

青い池スコアの計算

青い池の青は、水中のアルミニウムを含む微粒子(コロイド)が太陽光を散乱して生まれます。 だから理屈の上で色を左右するのは「水の濁り」と「差し込む光の量」と「水面の穏やかさ」の3つ。 スコアはこの3要素を点数化したものです。

基準点 55点からスタートし、以下を加減算して0〜100に丸めます。

1. 直近48時間の雨(濁り)— 最大の減点要因

直近2日の合計降水量加減点理由
30mm以上−35まとまった雨は川から濁った水を運び込み、青の源であるコロイドの見え方を壊します。「雨の翌日は白い」が定番のがっかりパターン
15〜30mm−20
5〜15mm−8
5mm未満+5

2. 日中(9〜15時)の平均雲量(光)

平均雲量加減点理由
30%以下+25青は光の散乱現象なので、日差しが強いほど鮮やかに出ます。曇天では同じ水でも灰色がかって見えます
30〜60%+10
60〜85%±0
85%超−10

3. 日中の平均風速(水面)

平均風速加減点理由
約6.9m/s以上−12風で水面が波立つと反射が乱れ、色がにごって見えます。立ち枯れの木の映り込みも消えます
約4.2〜6.9m/s−6
約4.2m/s未満+5

4. 今日の日中(6〜18時)の降水予報

予報降水量加減点理由
3mm以上−10見に行くその時間に雨だと、光も水面も条件が崩れるため

スコアと判定の対応

スコア判定
80〜100コバルトブルー期待
60〜79青が見えそう
40〜59五分五分
20〜39白濁ぎみかも
0〜19期待薄

Morning Fog Forecast

霧・雲海チャンスの計算

美瑛の丘の朝霧の多くは「放射冷却型」です。よく晴れた夜に地面の熱が逃げて空気が冷え、 湿った空気が飽和して霧になる——つまり「晴れた夜」「無風」「高い湿度」の3条件が揃った翌朝がチャンスです。 地元の写真家の経験則「翌日快晴の予報の朝、窓の外が真っ白なら丘へ急げ」を、そのまま点数にしています。

基準点 30点からスタートします(霧は「出ない日が普通」なので基準を低めにしています)。

1. 夜間(今日21時〜明日3時)の平均雲量

平均雲量加減点理由
30%以下+30雲がないほど地表の熱が宇宙へ逃げ(放射冷却)、明け方に空気が冷えて霧が生まれます。曇った夜は冷え込まず、霧も出ません
30〜60%+10
60%超−20

2. 明け方(3〜6時)の平均風速

平均風速加減点理由
約1.9m/s以下+20霧は静けさの中に溜まります。風が吹くと霧は流されて消えるか、そもそも形成されません
約1.9〜4.2m/s+5
約4.2m/s超−15

3. 明け方の平均湿度

平均湿度加減点理由
95%以上+25湿度が飽和(100%)に近いほど、わずかな冷え込みで水蒸気が霧に変わります
88〜95%+15
80〜88%+5
80%未満−10

4. 気温と露点の差(明け方)

気温 − 露点加減点理由
1.5℃以内+10露点は「その空気が飽和する温度」。気温が露点まで下がった瞬間に霧が発生するので、差が小さい=あと一歩の状態です

スコアと判定の対応

スコア判定
75〜100雲海チャンス大
55〜74霧の可能性あり
35〜54微妙なライン
0〜34期待薄

Wheat Season Meter

麦秋メーターの計算

小麦の生育は気温の積み上げにほぼ比例して進みます(農学でいう積算温度)。 麦秋メーターは、美瑛の日平均気温を4月1日(融雪・起生期の目安)から毎日足し合わせ、 過去10年の平年ペースと比較して「今年の丘が金色になる時期」を相対推定します。

手順

ステップ内容
1. 積算4/1からの日平均気温(0℃未満は0として)を毎日加算(単位: ℃・日)
2. 平年カーブ過去10年それぞれの積算カーブを平均し「平年ペース」を作る
3. ペース比較今年の積算値に平年が達する日付と比べて「◯日早い/遅い」を算出
4. ピーク予測平年の麦秋ピーク(7/20頃)を、そのズレぶんだけ前後させる

この方式を選んだ理由

品種や播種日ごとの必要積算温度(絶対値)を使う教科書方式は、校正データがないと±1週間は平気で外れます。 「平年よりどれだけ早いペースか」という相対比較なら、絶対値の不確かさを打ち消せるため、 初年度から実用的な精度が出せます。実際の麦秋日の記録が貯まったら絶対方式へ校正していく計画です。

※ データ: Open-Meteo Archive(白金青い池付近の推定日平均気温)。品種・播種時期・畑による差(±1週間以上)は吸収できません。

Limitations

このスコアの限界

  • 1地点の予報値です — 美瑛は丘ごとに天気が変わる土地。青い池付近の予報が、あなたの立つ丘と同じとは限りません。
  • 水の中は見ていません — コロイドの状態や上流の工事・雪解けの影響は気象データに現れません。特に雪解け期(4〜5月)は緑がかることがあります。
  • 係数は育成中です — 各点数は気象の定石と地元写真家の経験則から置いた初期値で、実際の見え方と照らして継続的に調整しています。

だからこのサイトは「予報」ではなく「条件充足度スコア」と呼んでいます。 確実を期すなら、出発前に美瑛町観光協会のSNSで最新の現地写真を確認するのが最強です。
